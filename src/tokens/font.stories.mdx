# Fonts

We use a combination of Fira Sans, Roboto Slab, and Ubuntu Mono for our font
stack. Each header level has it's own unique styles, so we do not provide CSS
or JS representation of these styles. We recommend you rely on the Vue
components provided in this package.

## Usage

This library contains Vue components for each type style. Simple drop them in
and use them like normal components. For more information on the props and slots
each provide, you can look under the "Components" section of the sidebar.

```vue
<template>
  <sys-header-1>This is header 1!</sys-header-1>
</template>

<script>
import { SysHeader1 } from '@system76/design'

export default {
  components: { SysHeader1 }
}
</script>
```
