## Usage

Obviously you can just copy the RGB values from this page into a project, but
to make it easier, we have included a number of files with all of these
variables for you. This ensures that the colors you want will always be up to
date. Everything you need is included in the npm `@system76/design` package.
Here are some examples of how to use it.

```js
import { colorWarmGray } from '@system76/design'

console.log(colorWarmGray) // rgb(87, 79, 74)
```

```css
@import "@system76/design";

body {
  background-color: var(--color-warm-gray);
}
```
